## 打印sql语句
1 . 一处打印
```
public function index()
{
    News::query()->get();

    DB::connection()->enableQueryLog();
    return response()->json(['log' => DB::getQueryLog()]);
}
```

2 . 生成路由
`php artisan route:scan`

3 . 请求接口 `curl 0.0.0.0:8081/simple`
